# Super Proxy Extension


Author: Roc Antonio

Description: Just a simpple server with a proxy to a given configuration to the file "config.proxy.js", can be placed anywhere.

config.proxy.js ex:

```
HOSTS: [
    {key: "asdf", endPoint: "http://asdf"}
]
```
